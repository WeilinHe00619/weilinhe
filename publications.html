Using Semantic Distance to Estimate Uncertainty in LLM-Based Code Generation

Weilin He, Arindam Sharma, Cristina David

Large Language Models (LLMs) show strong performance in code generation, but their outputs lack inherent correctness guarantees. We propose a semantic distance-based uncertainty estimation framework that clusters programs by execution behaviour and models inter-cluster distances to provide a reliable confidence score.
